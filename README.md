# 박스안에 머가 있을까?

---

### 1. 이 앱을 언제 사용하나요?
* 집안에 각 방 또는 창고에 정리된 박스 안에 머가 들어가 있는지 보고 싶을 떄 사용합니다.

### 2. 제약 조건은 무엇인가요?
* 박스 안에 들어갈 카테고리와 구성품들을 작성해주어야 합니다.

### 3. 구체적인 사례는 무엇인가요?
* 저희 집에서 자주 쓰지 않는 물건 또는 신발....등등을 박스에 넣어 보관하는데, 찾으려면 하나하나 꺼내봐야합니다.
* 찾고자 하는 물건이 어느 방 or 창고에 있는지 손쉽게 찾기 위해 사용합니다.
